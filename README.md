##  Install & Setup

**Option 1**

1.  Upload *dist* to server
2.  Configure */dist/en/index.html* or */dist/cn/index.html* as default page

**Option 2**

1.  `npm install`
2.  `gulp serve`
3.  Upload the latest *dist* to server
